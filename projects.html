Hecknet - Text-based computer & networking simulator

Hecknet is a small game with a functioning filesystem, and an unrealistic portrayal of cybersecurity and hacking. The structure of the game is meant to allow anyone to expand the program on a foundation that took some time before it was nailed down. The inspiration of Hecknet stems from my own personal fascination of programming and computer security- which is why I still attempt to keep this program updated to this day.

I would recommend checking it out.

Garfielf - Discord Chat Bot

The Garfielf chat bot started from a joke with friends- every Friday, we would send an image in our group chat of Garfield, with his owner, John, saying, "Friday again garfie baby". I chose to immortalize this tradition with the introduction of a chat bot that tells the user what day it is, and sends a message every Friday with that image.
